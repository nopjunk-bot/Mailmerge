📋 คำแนะนำการติดตั้งและใช้งาน

【ขั้นตอนที่ 1】ติดตั้ง Python
--------------------------------
1. ดาวน์โหลด Python จาก https://www.python.org/downloads/
2. เลือกเวอร์ชัน 3.8 ขึ้นไป
3. ติดตั้ง Python โดย **ต้องเลือก "Add Python to PATH"**
4. รีสตาร์ทคอมพิวเตอร์

【ขั้นตอนที่ 2】ติดตั้ง Library
--------------------------------
1. ดับเบิลคลิกที่ไฟล์ "install.bat"
2. รอจนกว่าการติดตั้งจะเสร็จ
3. กด Enter เพื่อปิดหน้าต่าง

【ขั้นตอนที่ 3】เตรียมไฟล์
--------------------------------
เตรียมไฟล์ 3 ไฟล์:
1. ไฟล์ Excel ข้อมูลนักเรียน (.xlsx)
2. ไฟล์ลายเซ็น (.jpg หรือ .png)
3. ไฟล์ฟอนต์ THSarabunNew.ttf

【ขั้นตอนที่ 4】รันโปรแกรม
--------------------------------
1. ดับเบิลคลิกที่ไฟล์ "run.bat"
2. โปรแกรมจะเปิดขึ้นมา

【ขั้นตอนที่ 5】ใช้งานโปรแกรม
--------------------------------
1. คลิก "เลือกไฟล์ Excel" → เลือกไฟล์ Excel ข้อมูลนักเรียน
2. คลิก "เลือกไฟล์ลายเซ็น" → เลือกไฟล์รูปลายเซ็น
3. คลิก "เลือกไฟล์ฟอนต์" → เลือกไฟล์ THSarabunNew.ttf
4. คลิก "เลือกโฟลเดอร์" → เลือกโฟลเดอร์ที่จะบันทึกไฟล์ PDF
5. คลิก "สร้างจดหมายเวียน"
6. รอสักครู่ จนกว่าโปรแกรมจะแจ้งว่า "สำเร็จ!"

========================================
💡 ข้อควรระวัง
========================================

1. ไฟล์ Excel ต้องมีรูปแบบเหมือนไฟล์ตัวอย่าง
   - มี header "เลขที่", "เลขประจำตัว", "คำนำหน้า", "ชื่อ", "สกุล"
   - มีคอลัมน์ภาคเรียน: 1/2566, 2/2566, 1/2567, 2/2567, 1/2568, 2/2568
   - ค่าที่จ่ายแล้วต้องเขียนว่า "จ่ายแล้ว"
   - ค่าที่ค้างชำระใส่เป็นตัวเลข

2. ลายเซ็นควรเป็นพื้นหลังสีขาว เพื่อให้สวยงาม

3. ฟอนต์ THSarabunNew.ttf ต้องเป็นไฟล์ฟอนต์จริง ไม่ใช่ฟอนต์ที่ติดตั้งในระบบ
